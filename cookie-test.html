üìã Implementation Details

‚úÖ Features Implemented:

  • First-Party Cookies: Uses persistent cookies instead of localStorage
  • 2-Year Expiry: Visitor ID cookies last for 2 years
  • Session Management: 30-minute session timeout with cookie-based tracking
  • GDPR Compliance: Opt-in/opt-out functionality
  • Security: SameSite=Lax, Secure flag on HTTPS
  • Privacy-First: No personal data stored, only anonymous IDs
  • No Fingerprinting: Uses simple, privacy-respecting approach

üîí Security Features:

  • SameSite=Lax prevents CSRF attacks
  • Secure flag on HTTPS prevents man-in-the-middle attacks
  • No HttpOnly flag (client-side accessible for opt-out)
  • No third-party cookies or cross-site tracking

üìà Metrics Tracked:

  • Unique visitors (via persistent visitor_id)
  • Returning visitors (detected via existing visitor_id)
  • Sessions (30-minute timeout)
  • Page views (automatic and manual)
  • Geographic data (IP-based, opt-out possible)
  • Device and browser information
  • Traffic sources and referrers
